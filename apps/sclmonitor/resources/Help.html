SCL Monitor Application


This application is for monitoring some SCL cavity waveforms.  As the first version, it can only monitor 2 high-order mode (HOM) waveforms, HB0 and HB1.  The “Take Snapshot” button on the tool bar allows users to save these waveforms, cavity rep rate and cavity field read-back for all SCL cavities to the PV Logger.  “Load from PV Logger” button pop-ups a PV Logger selector for users to choose a reference snapshot among the saved ones.  Once a reference PV Logger snapshot is selected, a red waveform will show on top of the existing live waveform, and corresponding table columns will also be filled.
