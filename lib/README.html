External Jar Files

Several features of Open XAL including functionality depend upon external (third party) jars. It should be stressed that these external jar files are not part of Open XAL. While Open XAL is distributed through BSD licensing on the Source Forge Open XAL project, many of the external jar files have different licensing terms (e.g. GPL, commercial) which restrict us from redistributing these files.

Please abide by the terms and conditions set forth by the third party vendors.
